# Introduction

Ajetreo is the design system powering all frontend projects under the [Ajetreo](https://ajetreo.co) umbrella.

# Get started

Ajetreo Design System (ADS) is a reusable component library that helps Storybook contributors build UIs faster. The goal is
to make building durable UIs more productive and satisfying.

## Install

ADS components are written in React, and its stories are written in
[Component Story Format](https://medium.com/storybookjs/component-story-format-66f4c32366df).

Add ADS to your project.

`npm install --save @ajetreo/design-system`

### **Use**

Import components you want into your UI

`import { Button, Badge } from '@ajetreo/design-system';`

and use them like so

```js
const example = () => (
  <div>
    <Button appearance='primary' onClick={() => 0}>
      Do something
    </Button>
    <Badge onClick={this.clear}>Cool</Badge>
  </div>
)
```

### **Browse ADS components in your own Storybook**

Once you add the package, update your .storybook/config.js to import all files ending in .stories.js.

```js
import { configure } from '@storybook/react'
configure(require.context('../src', true, /.stories.js$/), module)
```

### **Run and develop ADS locally**

Clone the [ADS GitHub project](https://github.com/ajetreo/design-system) then start Storybook.

`yarn && yarn run storybook`
